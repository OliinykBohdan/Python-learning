# Shopping cart (Python)

## Project Goal
The goal of this project is to build a simple command-line shopping cart system that allows users to manage products, track quantities, and calculate the total cost.

## About
This is a Python CLI application that simulates a small store.
Users can add products to a cart, remove them, and view the total price.
The program also keeps track of available stock and updates it dynamically.

## Features
- Display available products with price and stock
- Add products to the cart by weight (kg)
- Remove products partially or completely
- Clear the entire cart
- Automatic stock updates
- Calculate total price
- Input validation and error handling

## How to run
```bash
git clone https://github.com/OliinykBohdan/Learning-Python.git
cd Learning-Python/Projects/Console_Apps/Shopping_Cart
```
